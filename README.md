A 2D game built with bevy.

### Usage

Run it with the following:

```shell
cargo run --release
```

When developing, use this to dynamically link with bevy for faster builds:

```shell
cargo run --features bevy/dynamic
```

### Controls

- WASD keys to move
- Escape to exit

### Building

Build a release:

```shell
./build_release
```

Output will be `twodina.zip`.
